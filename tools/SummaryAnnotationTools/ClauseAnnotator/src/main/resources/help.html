General

Clause boundaries are bordered with handles: "[" and "]".

Sentences are separated with empty lines.

Markers are underlined.

Verbs are italic (by default works for Polish only). Verbs are identified by regular expression (with is compared to the pos tag of a word) stored in the file verb.regexp inside the jar package. Modify this file for your purposes using any extraction tool.

Marker/Clause/Span selection

When a span of text is selected, it's yellow. Program doesn't allow to select a span in more than one clause. Automatically this span's clause is selected. If the selected span contain only one marker (or part of a marker), this marker is also selected. Current selected clause/marker are displayed in the bottom panel. It's best to play with the program to see how it selects them.

All clauses which have set a "continue" attribute are highlighted pink.

Markers which don't have set required attributes (arg1/2, type) are marked magenta.

Clause actions

These are possible clause actions:

  • Split clause into two - select a single word or a span of words, then press ENTER key on the keyboard. The clause is going to be split in two, beginning from the first word selected. WARNING: System won't let you to split a marker into two clauses.
  • Merge clause with next - select clause, then press N key. Current clause is going to be merged with the next one, if possible. All pointers to current clause are going to be lost, all pointers to the next clause remain the same.
  • Merge clause with previous - select clause, then press P key. Current clause is going to be merged with the previous one, if possible. All pointers to current clause are going to be lost, all pointers to the previous clause remain the same.

Marker actions

These are possible marker actions:

  • Create marker - select a span of words, then press M key on the keyboard. Marker will be created from the selection.
  • Delete marker - select this marker, then press key D on the keyboard.
  • Change marker NUC attribute - select marker, then select desired NUC in the bottom panel.
  • Change marker CONNECT attribute - select marker, then select desired CONNECT in the bottom panel.
